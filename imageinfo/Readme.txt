			Image Informer is a simple program which shows basic information about image like Type of image, Height, Width, FileSize etc. In Image Informer you can see the image also. Image Informer only supports Jpg, Gif, Bmp and Png Files. All the Error handling is done but if u found any bug pls write to me at inderpal0@hotmail.com. You can easily understand the program.
